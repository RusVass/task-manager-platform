# Task Manager Web (Next.js)
Frontend for Task Manager API built with Next.js 14 (App Router), Tailwind, Server Actions, and JWT session via HTTP-only cookie.

## Quick start
```
cd apps/web
npm install
```

Create `.env.local`:
```
BACKEND_URL=http://localhost:3001
APP_BASE_URL=http://localhost:3000
COOKIE_NAME=tm_token
```

Run: `npm run dev` (port `3000`).  
Full stack/structure details live in the root `README.md`.
